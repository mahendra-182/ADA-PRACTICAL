# ADA Practical Programs

A collection of **Algorithm Design and Analysis (ADA)** practical programs implemented in Java.

## Programs List

| File | Topic | Algorithm |
|------|-------|-----------|
| PR1.java | Sorting | Selection Sort |
| PR2.java | Sorting | Bubble Sort |
| PR3.java | Sorting | Insertion Sort |
| PR4.java | Sorting | Merge Sort |
| PR5.java | Sorting | Quick Sort |
| PR6.java | Searching | Linear Search & Binary Search |
| PR7.java | Graph | Prim's Minimum Spanning Tree |
| PR8.java | Graph | Kruskal's Minimum Spanning Tree |
| PR9.java | Dynamic Programming | Coin Change Problem |
| PR10.java | Dynamic Programming | 0/1 Knapsack Problem |
| PR11.java | Dynamic Programming | Matrix Chain Multiplication |
| PR12.java | Dynamic Programming | Longest Common Subsequence (LCS) |
| PR13.java | Dynamic Programming | Shortest Common Supersequence (SCS) |

## How to Compile & Run

```bash
# Compile all files
javac PR1.java

# Run
java PR1
```

## Sample Outputs

- **PR1–PR5**: Sorting algorithms with random arrays and execution time analysis
- **PR6**: Search (Linear & Binary) on user-defined arrays
- **PR7**: Prim's MST — Edge list and total weight
- **PR8**: Kruskal's MST — Edge list and total weight  
- **PR9**: Minimum coins to make a target amount
- **PR10**: Maximum value fitting in knapsack capacity
- **PR11**: Minimum scalar multiplications for matrix chain
- **PR12**: Length and string of LCS between two sequences
- **PR13**: Length and string of SCS combining two sequences
